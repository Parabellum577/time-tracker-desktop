OAuth connect Gitlab (Self-hosted & Enterprise )

  1. Sign in to Gitlab
  2. Go to Settings -> Applications
  3. Name field - enter Application name (arbitrary)
  4. Redirect URI field -> paste https://web.timetracker.com/oauth
  5. Mark `api` scope and click `Save Application` button (Application ID and Secret key are created)
  6. Paste Application ID and Secret key into corresponding fields
  7. Click `Login via Gitab` button
